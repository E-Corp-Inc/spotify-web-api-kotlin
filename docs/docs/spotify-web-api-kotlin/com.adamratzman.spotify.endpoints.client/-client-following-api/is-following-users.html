isFollowingUsers

(common, js, jvm) fun isFollowingUsers(vararg users: String): SpotifyRestAction<List<Boolean>>

Check to see if the current user is following one or more other Spotify users.

Requires the SpotifyScope.USER_FOLLOW_READ scope

Api Reference

Parameters

users - List of the user Spotify IDs to check. Max 50

Exceptions

BadRequestException - if users contains a non-existing id

Return
A list of booleans corresponding to users of whether the current user is following that user
