isFollowingPlaylist

(common, js, jvm) fun isFollowingPlaylist(playlist: String, user: String): SpotifyRestAction<Boolean>

Check to see if a specific Spotify user is following the specified playlist.

Api Reference

Parameters

playlist - playlist id or uri

user - Spotify user id

Exceptions

BadRequestException - if the playlist is not found or if the user does not exist

Return
booleans representing whether the user follows the playlist. User IDs not found will return false
